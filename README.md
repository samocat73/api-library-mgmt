# API для управления библиотекой

## Этот сервис позволяет управлять библиотекой, а именно добавлять названия книг, авторов и прочее. Доступна регистрация пользователей, их авторизация. Сервис позволяет работать с отдельными группами, а именно с библиотекарями. Таким образом, можно разделять функциональность в зависимости от роли пользователя.

## Используемые технологии:
### - Django REST Framework
### - Django
### - JSON Web Tokens
### - PostgreSQL 
### - Docker 
### - Docker-Compose

## Проект настроен для запуска в контейнере Docker.

## Шаги для установки
### 1. Клонируйте репозиторий:
```
https://github.com/samocat73/api-library-mgmt.git
```
### 2. Заполните файл .env.example актуальными данными и переименуйте его в .env. Это нужно для корректной загрузки переменных окружения в проект!
```angular2html
SECRET_KEY=
DEBUG=

POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
HOST=postgres # ссылается на название сервиса postgres в файле docker-compose.yml
PORT=
```
### 3. Запустите сборку образов и контейнера с помощью команды:
```angular2html
docker-compose up -d
```
### Доступные для запросов эндпоинты вы можете посмотреть после запуска проекта по адресу:
```angular2html
http://localhost:8000/redoc/
```
### Для удобного заполнения базы данных данными вам будет удобнее использовать админку. Выполните команду и заполните стандартную форму:
```angular2html
docker-compose exec web python manage.py createsuperuser
```
### Теперь вам доступна панель администратора. Здесь вы можете управлять пользователями, назначать своих библиотекарей, которые смогут через запросы заполнять данные о книгах, авторах, а так же смогут видеть информация о выданных книгах, если выдадут их. Вся информация доступна в документации.
